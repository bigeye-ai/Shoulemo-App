# 瘦了么 (Shoulemo) - 體重不會騙人

這是一款受「死了么」啟發的極簡主義健康管理 App。它不提供溫柔的鼓勵，只提供冰冷的現實。

## 核心理念
> 「你可以不瘦，但你不能不面對。」

在「瘦了么」中，體重記錄不是一項功能，而是一場每日必須面對的審判。

## 核心功能
- **生死簽到**：今天瘦了嗎？沒瘦，也得填。
- **體重趨勢**：數據不會騙人，脂肪在顫抖。可視化圖表讓你直觀感受自己的放縱或堅持。
- **監督機制**：綁定監督人郵箱。如果你選擇逃避（不簽到），系統將自動向監督人「告狀」。
- **毒舌提醒**：高對比度黑綠設計，配以犀利的文案，時刻提醒你面對現實。

## App Store 描述文案建議
- **標題**：瘦了么 - 體重不會騙人
- **宣傳文本**：今天瘦了嗎？沒瘦，也得填。
- **詳細描述**：
  這不是一個溫暖的健身社區，這是一個冰冷的體重審判庭。
  
  我們相信，減肥失敗的核心原因不是缺乏方法，而是缺乏面對現實的勇氣。
  「瘦了么」強制你每天直面那個冰冷的數字。
  
  - 「今天瘦了嗎？」
  - 「沒瘦，也得填」
  - 「你可以不瘦，但你不能不面對」
  - 「體重不會騙人」
  
  如果你連續 24 小時未簽到，我們會把你的懶惰告訴你的監督人。
  數據不會騙人，脂肪在顫抖。

## 技術細節
- **Swift Charts**: 實現流暢的體重趨勢可視化。
- **SwiftData**: 本地存儲，保護隱私（你的體重只有你和你的監督人知道）。
- **SwiftUI**: 極簡、硬朗的黑綠配色 UI。
